Preconditions

noreferences

@@description

<p>You can specify basic preconditions in methods as shown below. This is still under development.</p>

@@syntax
[[methodBody]] [[precondition]] 

@@example
class PotentialVoter {
  Integer age;
  
  int vote(int candidate) {
    [pre: age >= 18]
    [pre: candidate > 0]
    // rest of stuff that we do not interpret
     return 0;
  }
}
@@endexample
