Below are instructions for downloading and installing Umple, a model-oriented programming technology that lets you model in code. Here is more information about Umple, or try it online without downloading, using the UmpleOnline tool

Requirements to run Umple

Umple can be used on Mac-OS, Windows or Linux.

For Eclipse

  • Indigo Eclipse with Modelling tools. The installation instructions below will also require you to install Jet and XText.

For command-line use

  • An up-to-date JVM.
 

Directions

Downloading Umple

  • The latest official releases of Umple can be obtained by clicking here. Both the command-line compiler and the Eclipse plugin are available.
     
  • The latest build of the command-line compiler from our continuous build process can be obtained by clicking here. This is also the version of the compiler used by UmpleOnline. This version will contain experimental features so should not be used in production.
     
  • Release notes: The list of completed changes applied in the last six months can be found by clicking here

 

Installing and running Umple in Eclipse

  • Follow the directions on this page to install Umple in Eclipse.

 

Running the Umple compiler on the command line

  • Run 
    java -jar umple.jar <options> <arguments>
  • Specify your umple file as an argument. 
  • You can use --help as an option to obtain the full set of command-line options. Options available in the continuous-build version of the command-line compiler include:
    • -g, --generate : Specify the output language: Java,Cpp, Php,Ruby,SQL,Ruby,Json,Ecore,TextUml, Yuml
    • --help : Display the help message
    • --override : If a output language is specified using option -g, ouptut will only generate language
    • --path : If a output language is specified using option -g, output source code will be placed to path
    • -v, --version : Print out the current Umple version number
